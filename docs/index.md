## ⚡ Core Flow

```
Teacher creates space
      ↓
Configures deliverable type (Individual / Group)
      ↓
Selects and invites specific students
      ↓
Students are notified and access the space
      ↓
   [Individual]              [Group]
Each student submits    One student submits
Teacher reviews each    Others are blocked + notified
Accept / Reject         Teacher accepts first submission
```
